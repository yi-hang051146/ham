# Ham - 个人主页

一个简洁、现代的个人主页，展示个人信息、技能和项目作品。

## 功能特点

- **响应式设计** - 完美适配电脑、平板和手机
- **深色模式** - 自动跟随系统主题
- **音乐播放器** - 在线播放背景音乐
- **课表显示** - 自动显示当前课程
- **思源笔记集成** - 展示笔记内容
- **关注内容卡片** - 分类展示感兴趣的内容

## 快速使用

### 在线访问

直接访问网站即可查看主页内容。

### 本地运行

如果你想在自己的电脑上运行：

1. 确保已安装 Python 3
2. 双击运行 `scripts/start-server.bat`（Windows）或 `scripts/start-server.sh`（Mac/Linux）
3. 浏览器会自动打开主页

## 主要功能说明

### 音乐播放器
- 点击播放按钮开始播放
- 自动循环播放歌单
- 记住你的播放状态

### 课表显示
- 根据当前时间自动显示课程
- 显示课程状态：进行中、即将开始、已结束
- 每分钟自动刷新

### 思源笔记
- 点击笔记本查看文档内容
- 支持数学公式和代码高亮
- 点击标题可以展开/折叠内容

### 关注内容
- 分类展示感兴趣的内容
- 点击卡片查看详细信息
